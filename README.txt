Nimble Multiprocessing Coding Challenge
Python 3.7.1

How to run from terminal: 
- Cd into 'nimblePythonChallenge/' directory
- Run 'python3 nimbleProgrammingChallenge.py'
- Enter amount of images, height, and width
- Either press enter to continue through images or 'q' to quit
Docstrings:
- Cd into 'nimblePythonChallenge/' directory
- Run 'python3 doc_nimbleProgrammingChallenge.py'
Pytest:
- Cd into 'nimblePythonChallenge/' directory
- Run 'pytest test_nimbleProgrammingChallenge.py'
